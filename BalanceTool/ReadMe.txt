GAMEPLAY INPUTS

ZQSD - Move
Space - Jump
Left click - Shoot
A - Use spell

EDIT INPUTS

Scroll wheel - Change camera moving speed
A / E - Go up / down

Right click and drag - Move camera


DATA INPUTS

F1 - Change tool state

F2 - Save data

F3 - Load data

F4 - Load Enemy data
